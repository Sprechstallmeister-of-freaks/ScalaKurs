# Первый файл Второй файл Схожесть Тип клона
Что означают типы клонов:
- T1 - отличается только форматирование (пробелы, переносы)
- T2 - переименованы переменные, но логика та же
- T3 - семантический клон: разная структура, но смысл одинаковый

Как это работает?

1. Вы указываете путь к папке с кодом
2. Программа находит все .scala файлы во всех подпапках
3. JPlag анализирует структуру кода (а не просто текст)
4. Вы получаете список похожих фрагментов
5. Чем выше процент схожести - тем вероятнее плагиат
